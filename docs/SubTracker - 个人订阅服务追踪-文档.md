## 专用名词

|                名词（Noun）                 | Description                                                | Notes      |
| :-----------------------------------------: | ---------------------------------------------------------- | ---------- |
|        订阅记录（Subscribe Record）         | 记录服务订阅情况的单位                                     | 简称记录   |
|        统计跨度（Statistical Span）         | 进行统计行为的时间窗口                                     |            |
|           预警时间（Alert Time）            | 某个订阅记录到期前的一段特定时间                           |            |
| 订阅记录时间轴（Subscribe Record Timeline） | 一个功能，以水平时间轴为载体，用色块表示每个订阅的持续时间 | 简称时间轴 |
